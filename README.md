# 💎 Radha Jewels — E-Commerce Website

A complete, responsive jewellery e-commerce website built with pure HTML, CSS, and JavaScript.

## 🌐 Live Demo
[View Live Site](https://your-site.netlify.app)

## ✨ Features
- 16 jewellery products across 10 categories
- Live search with instant results
- Shopping cart with coupon codes
- Wishlist sidebar
- Product quick-view modal with size selector
- Real checkout form with validation
- Order confirmation screen
- Skeleton loading animations
- Toast notifications
- Cookie consent banner
- Fully responsive (mobile/tablet/desktop)
- Free shipping progress bar

## 🛠️ Built With
- HTML5
- CSS3 (CSS Variables, Flexbox, Grid, Animations)
- Vanilla JavaScript (No frameworks)
